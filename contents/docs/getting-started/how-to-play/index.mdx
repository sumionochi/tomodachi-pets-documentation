## ğŸ… Bonus Step: Walrus Integration â€“ Decentralized Storage

Tomodachi Pets uses **Walrus** by Mysten Labs for all image storageâ€”making every NFT truly decentralized and Web3-native.

1. **Image Upload**  
   When your accessory or pet image is finalized (drawn, uploaded, or AI-generated), itâ€™s sent to a Walrus Publisher node via the `/api/upload` route as binary data.

2. **Content Addressing**  
   Walrus returns a <code>blobId</code> (unique hash for your image). The app forms a URL like:  
   <code>https://blobs.testnet.walrus.network/&lt;blobId&gt;</code>  
   This is stored in the NFTâ€™s on-chain metadata.

3. **On-Chain Reference**  
   When you mint your accessory NFT, the Walrus URL is saved in the `url` field of the Asset object (on-chain).  
   Anyone can retrieve the image from Walrus using this link.

4. **Benefits**  
   - No central server: Images are stored on a decentralized network.
   - Always available: Any user can fetch the image with the link.
   - On-chain ownership: Image link is part of the NFT object itself.
